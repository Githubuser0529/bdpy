# BdPy

Python library for brain decoding

## Requirements

- numpy
- scipy
- h5py

### Optional requirements

- 'dataform' module
    - pandas
- 'mri' module
    - nipy

## Modules

- bdata
- dataform
- ml
- mri
- preproc
- util

## For developers

Please send your pull requests to `dev` branch, not to `master`.
